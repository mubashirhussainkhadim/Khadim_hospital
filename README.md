# Khadim Hospital Nuxt Website

This workspace contains a Nuxt 3 starter website for **Khadim Hospital** in **Kot Radha Kishan, Kasur, Punjab, Pakistan**.

## Included

- Nuxt 3 project scaffold
- Responsive homepage
- SEO metadata
- Hospital sections for services, doctors, and contact details

## Public information used

The initial content was based on public web information available on:

- `https://khadimhospital.netlify.app/`
- `https://kasur.punjab.gov.pk/district%20_profile`

Before production launch, verify the doctor list and phone number with the hospital directly.

## Run locally

Node.js is not installed in this environment, so dependencies could not be installed here. On a machine with Node 20+:

```bash
npm install
npm run dev
```

Then open the local Nuxt URL shown in your terminal.
